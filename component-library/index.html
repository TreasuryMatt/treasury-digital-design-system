Buttons

All buttons use the .usa-button base class plus a variant modifier. Base styles: 15px, 700 weight, 2px border, 4px radius, 10px / 20px padding.

Variants

<!-- Primary action -->
<button class="usa-button usa-button--primary">Save</button>

<!-- Secondary / neutral -->
<button class="usa-button usa-button--secondary">Cancel</button>

<!-- Outlined -->
<button class="usa-button usa-button--outline">Export</button>

<!-- Confirm / approve -->
<button class="usa-button usa-button--success">Approve</button>

<!-- Destructive action -->
<button class="usa-button usa-button--danger">Delete</button>

<!-- Link-style, no border or background -->
<button class="usa-button usa-button--unstyled">View details</button>

Small Buttons — .usa-button--sm

Reduced to 13px / 6px–14px padding. Use inside tables, card headers, toolbars, and filter bars.

Toast — .toast

Fixed bottom-center notification. Auto-dismisses after 4 s. Use role="status" + aria-live="polite" so screen readers announce it without interrupting. Never stack toasts — each call replaces the current one. Use an inline .usa-alert instead for errors requiring user action.

<!-- HTML (rendered by JS) -->
<div class="toast toast--success" role="status" aria-live="polite">
  <span>Changes saved successfully.</span>
  <button class="toast__dismiss" aria-label="Dismiss">&times;</button>
</div>

<!-- Vanilla JS helper — add once per app -->
<script>
let _toastTimer = null;
function showToast(message, type = 'success') {
  let el = document.getElementById('tdds-toast');
  if (!el) {
    el = document.createElement('div');
    el.id = 'tdds-toast';
    el.setAttribute('role', 'status');
    el.setAttribute('aria-live', 'polite');
    document.body.appendChild(el);
  }
  el.className = `toast toast--${type}`;
  el.innerHTML = `<span>${message}</span>` +
    `<button class="toast__dismiss" aria-label="Dismiss" onclick="dismissToast()">&times;</button>`;
  clearTimeout(_toastTimer);
  _toastTimer = setTimeout(dismissToast, 4000);
}
function dismissToast() {
  const el = document.getElementById('tdds-toast');
  if (el) el.remove();
  clearTimeout(_toastTimer);
}
</script>

Sidebar Navigation

The USWDS side navigation pattern for authenticated app shells. A sticky left rail (240px) organizes primary navigation into labeled groups. The left-border indicator marks the active link — usa-current on the active <a>.

Basic Sidebar — .usa-sidenav

Wrap the list in .usa-sidenav-container and use .usa-sidenav__item on each <li>. Add usa-current to the active <a>.

  • Dashboard
  • Projects
  • Reports
  • Users
  • Settings
<nav class="usa-sidenav-container" aria-label="Primary navigation">
  <div class="sidenav-section">
    <ul class="usa-sidenav">
      <li class="usa-sidenav__item">
        <a href="/dashboard" class="usa-current">Dashboard</a>
      </li>
      <li class="usa-sidenav__item"><a href="/projects">Projects</a></li>

</nav>

Dark Mode

TDDS dark mode is implemented entirely via CSS custom property overrides on [data-theme="dark"]. Since every component reads var(--usa-*) tokens, toggling the attribute on <html> propagates to the entire app with no per-component changes. Use the toggle in the sidebar to preview.

Theme Toggle Button

Drop this button into your site header or toolbar. It toggles data-theme="dark" on <html> and persists the choice to localStorage. On first load it respects the OS preference via prefers-color-scheme.

Live demo — this is the actual toggle powering this page:

<!-- HTML -->
<button class="usa-theme-toggle" onclick="toggleTheme()" aria-label="Toggle color theme">

/* JavaScript */
function toggleTheme() {
  const isDark = document.documentElement.getAttribute('data-theme') === 'dark';
  const next = isDark ? 'light' : 'dark';
  document.documentElement.setAttribute('data-theme', next);
  localStorage.setItem('tdds-theme', next);
  document.querySelectorAll('.usa-theme-toggle__label')
    .forEach(el => { el.textContent = isDark ? 'Dark' : 'Light'; });
}

/* On page load — respect localStorage, then OS preference */
(function () {
  const saved = localStorage.getItem('tdds-theme');
  const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
  const theme = saved || (prefersDark ? 'dark' : 'light');
  if (theme === 'dark') {
    document.documentElement.setAttribute('data-theme', 'dark');
    document.querySelectorAll('.usa-theme-toggle__label')
      .forEach(el => { el.textContent = 'Light'; });
  }
})();
